Training our bot:
Choose which map to train on or random
Choose number of test runs
Press Run Agent Training

To start a fresh training session:
Delete the weights file
Restart the program (to re-seed random counters)

Disclaimer:
When the bot is first trained after removing weights a file not found error occurse, ignore this.

Running a trained bot:
Have a weights file
Choose map
Press New Game
Press Run Solving Agent until he finds the gold
